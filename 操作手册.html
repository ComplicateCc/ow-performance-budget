POI 区域性能预算工具 · 操作手册

单区域 POI 性能规划与热力评估

本工具用于在最多 2000m × 2000m 的单个规划区域内，评估 POI、默认环境层、LOD/HLOD、遮挡剔除和 Android 画质预算之间的关系。 它不是完整大世界编辑器，而是面向区域 POI 布局和实时性能预算的本地浏览器工具。

1. 启动方式

  1. 进入项目目录：C:\Projects\AI\CodeX\ow-performance-budget
  2. 安装依赖：npm install
  3. 启动开发服务：npm run dev -- --host 127.0.0.1 --port 5174
  4. 浏览器打开：http://127.0.0.1:5174/

如果 5173 已被其他项目占用，请继续使用 5174 或换一个空闲端口。

2. 界面结构

左侧面板

配置默认层、POI 列表、POI 参数、物件模板、区域大小、Android 画质预算。

中央地图

显示区域网格、POI 矩形、相机视锥、LOD 可视范围和网格热力图。

右侧面板

查看实时 DP/Triangles、分类统计、POI 统计、模板模拟值、相机参数、数据导入导出和版本对比。

顶部栏

快速切换热力图开关，以及在 DP 和 Triangles 两种热力指标之间切换。

3. 推荐工作流

  1. 先在“区域与 Android 画质”里设置区域宽高、地块大小、画质等级和预算。
  2. 配置“默认层”，让非 POI 区域具备基础环境负载。
  3. 在地图上双击创建 POI，或用左侧 POI 管理面板新增、复制、删除 POI。
  4. 选中 POI 后，调整位置、尺寸、等级、遮挡剔除率和物件数量。
  5. 调整物件模板的 LOD/HLOD/消失距离，观察右侧预算变化。
  6. 打开热力图，切换 DP/Triangles，检查不同地块和方向的压力分布。
  7. 保存项目 JSON 或导出 HTML 报告，用快照功能记录阶段版本。

4. 地图操作

操作 说明
双击地图 在点击位置创建一个默认 M 级 POI。
点击 POI 选中 POI。若多个 POI 重叠，会优先选中面积较小的 POI。
拖动 POI 移动 POI 位置，右侧性能数据会实时刷新。
拖动 POI 右下角 调整 POI 尺寸。
拖动相机点 移动视锥采样位置。
拖动相机朝向手柄 旋转视角方向。
滚轮或缩放滑杆 缩放画布视图，不改变实际区域尺寸。

5. 默认层

默认层覆盖整个区域里的非 POI 部分，结构与 POI 一致，包含等级、遮挡剔除率和各类物件数量。 计算时它会排除所有 POI 占用区域，避免默认层和 POI 重复统计。

默认层适合放置基础植被、散布小物件、远景环境等“非重点 POI”的背景负载。

6. POI 管理

  • 等级：支持 XL/L/M/S，影响默认创建尺寸和视觉颜色。
  • 遮挡剔除率：POI 自身剔除比例，默认 50%。实际计算还会乘以全局剔除系数。
  • 位置和尺寸：支持面板输入，也支持地图拖动。
  • 物件数量：每个 POI 可配置各物件模板的聚合数量，不逐个摆放单体资产。
  • 复制/粘贴/副本：可用按钮，也支持常见快捷键复制粘贴选中 POI。

7. 物件模板、LOD 与 HLOD

每种物件类型都有基础 DP、基础 Triangles、3 级 LOD、HLOD 和消失距离。 左侧模板面板负责定义规则，右侧“模板模拟值”面板可以直接修改定值，百分比会自动同步。

字段 含义
LOD DP% 对应 LOD 下的 Draw Call 百分比。默认 3 级均为 100%。
LOD Tri% 对应 LOD 下的面数百分比。默认 100%、50%、30%。
HLOD 距离 超过该距离后使用 HLOD 的 DP/Tri 百分比。
消失距离 大于 0 时，超过该距离后不再计入预算，也不会进入 HLOD。

8. 热力图

热力图使用网格采样。每个地块在中心点分别向上、右、下、左四个方向模拟一次视锥预算，然后用四个三角区域显示结果。 地块大小默认 64m × 64m，可在左侧区域设置中调整。

  • DP/Triangles：顶部下拉框切换热力图指标。
  • 热力透明：控制整体覆盖强度，方便未来叠加地图背景。
  • 低负载淡化：低负载区域会降低透明度，避免整图都是绿色时遮挡网格。
  • 网格置顶：热力图绘制在网格下方，打开热力图后仍能看清网格线。

9. 性能计算逻辑

  1. 根据当前相机视锥判断 POI 或默认层采样点是否可见。
  2. 按采样点距离选择 LOD 或 HLOD。
  3. 应用物件模板的基础 DP/Triangles 与百分比。
  4. 应用 POI 或默认层遮挡剔除率，再乘全局剔除系数。
  5. 汇总分类、POI 和总量，与当前 Android 画质预算对比。

11. 数据与报告

  • 保存项目：导出完整项目 JSON，包括区域、默认层、POI、模板、画质配置和版本快照。
  • 加载项目：导入项目 JSON。导入前会校验格式，失败时保留当前工作。
  • 模板导入/导出：只保存或加载物件模板。
  • 画质导入/导出：只保存或加载 Android 画质预算配置。
  • HTML 报告：导出当前性能结果，可用浏览器打印或另存为 PDF。
  • 截图：导出当前地图区域截图。
  • 版本快照：保存阶段方案，并在右侧面板进行对比。

12. 注意事项

  • 本工具只针对单个规划区域，最大支持 2000m × 2000m。
  • POI 内物件按聚合数量计算，不代表真实单体摆放。
  • 热力图是采样模拟结果，用于发现压力区域和方向差异，不等同于最终引擎 Profiler 数据。
  • 数字输入框会在失焦或回车后提交，输入过程中不会立刻强制跳到最小值。
